# Plan de Implementación Maestro - Ravyn

Este documento detalla las fases para la evolución de Ravyn, desde su base actual hasta la plataforma interactiva y escalable proyectada.

## Fase 1: Modernización Técnica (Completada Lógica / Pendiente Estética)
- [x] Inicialización con Vite + React + TypeScript.
- [x] Reorganización de archivos estáticos (public/).
- [x] Definición de tipos globales (Interfaces de pedidos).
- [x] Implementación de `LienzoRavyn` (Orquestador de módulos).
- [x] Migración modular de componentes (Historia, Trivia, Tarjetas, etc.).

## Fase 1.5: Ecosistema Ravyn (Landing & Showcase)
- [x] Definición de Identidad Visual (Brand Guidelines).
- [x] Estructura de la Landing Page (Hero, Secciones de Venta).
- [x] Navegación de Showcase (Catálogo de módulos interactivos).
- [x] Sistema de Configuración Previa (Sincronización global de temas).

## Fase 2: Experiencia de Usuario y Temas
- [x] Auditoría y Refinamiento CSS (Paridad visual total en todos los temas).
- [x] Implementación del sistema dinámico de temas (CSS Variables & ThemeManager).
- [ ] Refinamiento de animaciones y transiciones (Framer Motion).
- [ ] Optimización de carga de activos (lazy loading de imágenes y videos).

## Fase 3: Ravyn Store (Showroom Táctil)
- [ ] Implementación del Feed de Packs (Cartas de rareza con efectos visuales).
- [ ] Desarrollo del Modal de Previsualización (Live Demo con iframe).
- [ ] Construcción del Carrito Visual (Maqueta vertical de módulos).
- [ ] Integración de cotización para planes personalizados.

## Fase 4: Automatización y Backend
- [ ] Integración con n8n para procesamiento de pedidos.
- [ ] Implementación de lógica de IA para generación/curación de contenido.
- [ ] Sistema de generación de links únicos.

## Fase 5: Expansión Premium
- [ ] Módulo de Rompecabezas Interactivo.
- [ ] Wrapped VSoon (Estadísticas avanzadas).
- [ ] Soporte para dominios personalizados.
